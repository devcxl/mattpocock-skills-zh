# Triage Labels（分类标签）

技能使用五个标准化的分类角色。本文件将这些角色映射到本仓库 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本 tracker 中的标签 | 含义 |
| -------------------------- | -------------------- | ---------------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估该 issue |
| `needs-info`               | `needs-info`         | 等待报告人提供更多信息 |
| `ready-for-agent`          | `ready-for-agent`    | 已完全指定，可供 AFK agent 处理 |
| `ready-for-human`          | `ready-for-human`    | 需要人类实现 |
| `wontfix`                  | `wontfix`            | 不会被处理 |

当技能提到某个角色时（例如"应用 AFK-ready 分类标签"），请使用本表中对应的标签字符串。

编辑右侧列以匹配你实际使用的词汇。
